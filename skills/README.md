# Skills for Design Engineers

This directory contains design and animation skills from [Emil Kowalski's skills repository](https://github.com/emilkowalski/skills).

These skills encode taste-driven design engineering principles for building interfaces that feel great. They are used by AI agents to evaluate and improve UI quality.

## Included Skills

| Skill | Purpose |
|-------|---------|
| **emil-design-eng** | Core design engineering philosophy — animation decisions, easing, component polish |
| **review-animations** | Strict animation code review against craft standards |
| **improve-animations** | Audit all animations in a codebase with prioritized plans |
| **find-animation-opportunities** | Find places in the UI that genuinely benefit from motion |
| **apple-design** | Apple's interface design principles (WWDC design talks) translated for the web |
| **animation-vocabulary** | Glossary mapping vague motion descriptions to exact terms |
